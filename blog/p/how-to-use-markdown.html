# 使用 Markdown 排版文章

——图文排版是集合计算机算法和软件工程等学科的一门艺术。

一些排版软件

  • Office Word 和各种在线文档是所见即所得的，但是并不方便开源行业使用，而且功能过于臃肿，不简洁高效和统一优雅
  • 是算法和程序设计技术先驱 D.E. Knuth 发明的排版系统，对 PDF 格式支持更好，是学术论文排版的首选，但是编译时间长，非所见即所得，在新窗口打开 是 最广为人知的衍生
  • Doxygen 是开源并跨平台的文档生成系统，但是学习门槛较高，泛用性可能没那么好

更多情况，请选择 Markdown

Markdown 是简单高效的文章排版标记方式，同样所见即所得，被互联网广为使用，有大量工具软件支持，简单通用，一次编写到处发布

注意

一些包含代码、公式和框图等例子，仅仅只是演示 Markdown 排版和展示内容的能力，你只需要关注 Markdown 的标记语法起到的排版、布局和改变样式的作用

# 多级标题

语法

使用 N 个 # 空格后接标题

提示

Markdown 相当于对 HTML 的简化， Markdown 通常要先转换为 HTML ，然后再被排版渲染

<h1>一级标题</h1>
<h2>二级标题</h2>
<h3>三级标题</h3>
<h4>四级标题</h4>
<h5>五级标题</h5>
<h6>六级标题</h6>
查看 Markdown 代码
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

# 多行代码

提示

就像分隔线一样，更多的反引号 ```` 或者波浪号 ~~~~ 同样是可以的

注意

正确的三个反引号是 ``` ，而不是 ˋˋˋ ，它们只是看起来很像而已

#include <stdio.h>

int main() {
	puts("Hello, World!");
	return 0;
}
查看 Markdown 代码
```
#include <stdio.h>

int main() {
	puts("Hello, World!");
	return 0;
}
```

# 高亮代码

int main() {
	puts("Hello, World!");

查看 Markdown 代码

#include <stdio.h>

int main() {
	puts("Hello, World!");
	return 0;
}
```

class Program {
	static void Main(string[] args) {

	}
