使用Python统计历年来各种英语高考题中单词的出现频数

发表于 2019-02-09 | 更新于 2019-02-10

我从网上搜集了历年来高考英语新课标卷真题，还有各个省的高考真题，以及各种模拟题，题型汇总总共20多个doc文件，然后使用正则匹配删除了除了英文单词外的所有字符（保留空格），使用Python写了一个很简单的程序，统计出了8千多个不同的单词及其频数，使用Excel进行排序。
