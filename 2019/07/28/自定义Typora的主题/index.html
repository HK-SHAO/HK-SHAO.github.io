自定义Typora的主题

发表于 2019-07-28

Typora是一款MarkDown编辑器，界面简单美观，功能也很到位，数学公式可以完美显示，还可以直接用代码画树状图流程图，排版很简单，用起来特别爽。

但是，用了一段时间后感觉默认github主题的字体比较丑，中文字体没有力度，英文字体不够硬朗，尤其是字母“I”和“l”无法区分，符号’和‘也无法区分，所以自己就修改了一下主题的字体配置，搞了一个新主题。

在文件->偏好设置->打开主题文件夹，新建个new.css，代码如下

# 实用工具 # 教程 # 软件 # PDF # Office
英语语法
