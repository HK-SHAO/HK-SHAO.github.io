收藏的网站集合

这里是我收藏的一些网站，本页不定期修改和更新，如果你感兴趣，可以收藏本页 :)

如果你有推荐的有趣网站，可以在评论区分享哦！


Blender（布兰德）永久中文服务站！ (bgteach.com) 在这里下载最新版 Blender 比较快

iData-知识检索 (cn-ki.net) 搜索和下载中外文献的好工具

首页 - 萌番组 (bangumi.moe) 追番网站，在这里你可以搜索下载各个字幕组发布的番剧种子

全国图书馆参考咨询联盟 (superlib.net) 这里你能搜索到图书、期刊、报纸、专利、标准、音视频和报告

Jiumo Search 鸠摩搜索 - 文档搜索引擎 (jiumodiary.com) 搜索图书、文档

Ubuntu Pastebin 贴代码或者其它任何文本的工具

实践教学平台 (educoder.net) 编程教学平台

DeepL翻译：全世界最准确的翻译 翻译的准确度比较好，就是有点慢

Matrix67: The Aha Moments 知名博客，写了很多有意思的文章，可惜现在不更新了

在线LaTeX公式编辑器-编辑器 (latexlive.com) 是我认为最好用的 LaTeX 公式编辑器

Shadertoy BETA 着色器（shader）分享网站

The Book of Shaders shader 的教程，写的很详细，通俗易懂，可惜现在都还没更新完。

WebGL Fluid Simulation (paveldogreat.github.io) 一个用 Javascript 和 WebGL 实现的流体模拟器，效果非常炫酷

Graph Editor (csacademy.com) 绘制无向图或者有向图的工具

清华大学开源软件镜像站 | Tsinghua Open Source Mirror 开源软件镜像站

图形计算器 (desmos.com) 非常好用的图形计算器，功能没有 Geogebra 多，但是设计的比较出色

ConvNetJS: Deep Learning in your browser (stanford.edu) 斯坦福大学搞的 Javascript 版本卷积神经网络框架

JSFuck - Write any JavaScript with 6 Characters: ]()!+ 将一切 Javascript 代码转化为等价的 6 个字符，并且可以被 Javascript 解释器正确解释

免费3D模型 - Free3D.com 能下载各种各样的 3D 模型

A Neural Network Playground (tensorflow.org) 谷歌用 TensorFlow.js 搞的神经网络游乐场

Jez Swanson - itch.io 国外大佬搞了一些浏览器小游戏

OI Wiki (oi-wiki.org) 信息学竞赛的维基百科，在这里你能学到很多算法知识

AcWing 一个 OJ 平台，有趣的地方在于有段位和同台竞技系统

数据集 - 超神经 (hyper.ai) 能下载数据集

秘塔写作猫 (xiezuocat.com) 辅助中文写作，能自动分析文章是否有错误，并给出改正意见

Line Rider 一个画线让滑雪小人滑雪的游戏，游戏效果和流畅度都非常高

Tero Parviainen 一个大佬的个人主页，他研究并制作了一些用 AI 的方式产生音乐的有趣玩意

JS1k - The JavaScript code golfing competition 1 Kib Javascript 代码能干出哪些有趣的事？

iBug - iBug 大佬的博客，他曾经用 Javascript 写了一个生成 Mathematica 密钥的工具

报告查一查 (seedsufe.com) 能查到各种各样的报告，有参考价值，写文档或者做 PPT 的时候能用到

独立精神 | indienova 独立游戏 优质的独立游戏平台，在这里你能够发布自己的独立游戏

Tokyo Toshokan :: #tokyotosho @ irc.rizon.net :: Torrent Listing 在这里你能搜索到几乎所有日本的 BT 资源

始终 (liam.page) 一个大佬的博客，他写的计算机科学文章都非常有含量，比如《谈谈梅森旋转：算法及其爆破》

片库 - 全网高清影视聚合下载 (pianku.li) 最全看片平台，几乎没什么恼人的广告

简介 - CTF Wiki (ctf-wiki.org) 计算机网络安全（CTF）的维基百科，在这里你能学到很多黑客技术

计算机图形学与混合现实研讨会 – GAMES: Graphics And Mixed Environment Seminar (games-cn.org) GAMES的官网，计算机图形学的内容

Wolfram|Alpha: Computational Intelligence (wolframalpha.com) Mathematica 的超集和在线版

Material Design Icons 能下载很多 MD 设计的图标，在制作 UI 的时候可以用到

Icon图标在线制作工具 (p2hp.com) 非常简便的制作一个 MD 风格的图标

MaterialTree (mecg.me) 大佬自己手撸了一个在线版虚幻引擎蓝图风格的材质编辑器

Inigo Quilez :: fractals, computer graphics, mathematics, shaders, demoscene and more (iquilezles.org) 国外大佬的个人主页，主要是计算机图形、分形、shander 等内容

Keith Clark: Front-end web developer 国外大佬的个人主页，他用前端技术搞了很多有趣的东西

Danbooru: Anime Image Board 无穷无尽的二次元图片

📙 Emojipedia — 😃 Home of Emoji Meanings 💁👌🎍😍 能下载到不同设备和系统的 emoji 表情

Library Genesis (libgen.rs) 能搜索到几乎所有文献

Red Alert 2: Chrono Divide 红色警戒 2 的在线版，国外大佬自己用 typescript 把这个游戏复刻了，几乎一模一样

语音维基 (voicewiki.cn) 各种游戏里面语音的维基百科，比如 Ra2、MC、LOL 等

Apache ECharts 前端图表工具，非常强大

GLSL Editor (thebookofshaders.com) 在线的 GLSL Shader 编辑器

科学空间|Scientific Spaces 国内大佬的博客，很有水平

The On-Line Encyclopedia of Integer Sequences® (OEIS®) 数列索引网站

Ping.cn:网站测速-ping检测-dns查询-ipv6网站测试-路由跟踪查询 ping 一个网站，检测中国各个地区和不同类型网络能否正常访问这个网站

LearnOpenGL CN (learnopengl-cn.github.io) 学习 OpenGL 的文档

Try It Online (tio.run) 能在线运行几乎所有编程语言

JSLinux (bellard.org) 在浏览器上运行各种操作系统（是用本地的算力运行，作者用 Javascript 搞了个模拟器）

如果你有推荐的有趣网站，可以在下面分享哦！
